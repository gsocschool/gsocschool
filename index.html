    Google Summer of Code

    Google Summer of Code is an internship program offered by Google every year for Bachelor, Masters, Ph.D students over the globe. It gives them the opporunity to work full time in their summer in an open source project and get mentored by top-notch developers. It also gives them stipends to support them during the program, $5500 which is awesome!

    Get Going —

    You just need to attend our workshops, study and work hard till the program starts to get a better chance of getting selected

    You will also need a Text Editor, if you don't have one already, you may want one for editing code. A few options: Atom, Sublime Text, Textmate, Brackets.

    Core

    These workshops focus on essential skills for applying to Google Summer of Code.

    Stuck? Ask a question in the discussion.

    Introduction

    Get introduced to the Google Summer of Code program, how it works, and how to apply. Get to know each others, and mentors will write each atendee skills for the next matching workshop

    Learn Git

    To teach Git, and Github basics which is necessary for any Open Source project.

    sudo apt-get install git...

    Skills Matching

    In this workshop we will suggest some Open Source organizations to the atendee from the skills he/she gave to us in the introductory workshop.

    lets start hacking!

    Writing a Proposal

    Learn to write a good and eye catching prposal, the building blocks of it, and what mentors search for. Also we will provide some awesome accepted proposals from previous students.

    Electives
